rCharts

Interactive Visualizations from R

Ramnath Vaidyanathan
McGill University

rCharts helps create, customize and share dynamic, interactive visualizations from R, using a consistent lattice-like plotting interface.

Quick Start

library(devtools)
pkgs <- c("slidify", "slidifyLibraries", "rCharts")
install_github(pkgs, 'ramnathv')

rCharts helps create, customize and share dynamic, interactive visualizations from R, using a consistent lattice-like plotting interface.

 
library(rCharts)
rPlot(mpg ~ wt | gear, data = mtcars, type = 'point')

nPlot(mpg ~ wt, group = 'gear', data = mtcars, 
  type = 'scatterChart'
)

Leaflet Map

map3 <- Leaflet$new()
map3$setView(c(51.505, -0.09), zoom = 13)
map3$tileLayer(
  provider = "Stamen.Watercolor",
  maxZoom = 18 
)
map3$marker(c(51.5, -0.09), bindPopup = "<p> Hi. I am a popup </p>")
map3$set(height = 590, width = 950)
